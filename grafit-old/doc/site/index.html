about grafit

Data analysis and graphing software.

  • Line and point plots, postscript output via gri.
  • Nonlinear curve fitting
    • fast (uses odr)
    • interactive placing of fit functions
    • user defined functions in Python.
  • Extensible with plug-ins in python.
  • All internal functions available from the interactive Python console.
  • All functions in the extensive scientific library scipy are available.
  • Complete and unlimited undo/redo.

Grafit is free software, released under the GNU General Public License.

Grafit is written in Python using PyQt and PyQwt.
