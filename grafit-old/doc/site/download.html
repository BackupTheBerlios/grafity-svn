download grafit

You can download grafit here.

There is also a Subversion repository at http://svnhosting.org:8000/svn/grafit/.

You need to have a few other things to make it work:

  • Python (2.3 or later)
  • PyQt and PyQwt
  • A fortran compiler to compile odr
  • gri (for postscript graphs)

Some time soon I hope to clean up the code, write some instructions, and release a beta version.
